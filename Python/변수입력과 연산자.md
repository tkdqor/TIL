## 변수입력과 연산자
```python
a = input("숫자를 입력해주세요 : ")
print(a)
>>> 숫자를 입력해주세요 :      -> 여기서 숫자 5를 입력하면
>>> 5
```

```python
a, b = input("숫자를 입력해주세요 : ").split()
print(a, b)
>>> 숫자를 입력해주세요 :      -> 여기서 숫자 2개를 띄어쓰기로 입력하면 ex) 3 5
>>> 3 5
```
- split() 함수는 띄어쓰기를 분리자로 해서 입력값을 분리하고 a와 b 각각에 넣게 된다.
- input() 함수는 숫자를 받게 되어도 문자열로 값이 반환된다.
